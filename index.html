Bienvenido

Este portal ofrece fotos de Colombia, georeferenciadas indicando la orientación a la cual fueron tomadas.

El portal se encuentra bajo el directorio phpmygph: /phpmygpx/phpmygpx
Sin embargo, hay un acceso a la lista completa de fotos que coordinado por phpgraphy
Si no, se puede acceder por medio de Gallery.
El mapa sobre el cual se muestran las fotos es OpenStreetMap. Para ver una comparación de este mapa con su competencia, por favor visitar esta página http://mapcomparator.gmapify.fr/

Fotos

Las fotos son de libre acceso y están licenciadas bajo Creative Commons - Shared Alike. Para más información sobre la licencia visitar la siguiente página Atribución Compartir-Igual 2.5 Colombia

Todas las fotos tienen la licencia incluída en su encabezado EXIF y como parte de la presentación de las fotos se incluye un RDF que también contiene la licencia.

Servicio de obtención de fotos según criterios

Adicionalmente a las fotos y la base de datos de meta datos, se provee un servicio de consulta de fotos según criterios de búsqueda. Este servicio también es abierto
Esto cumple con la definición de servicio abierto, ya que el programa base que se usa por debajo es phpMyGPX que tiene licencia GPL, además, este hace referencia al proyecto abierto OpenStreetMap, por medio de OpenLayers; las fotos son libres bajo CC-SA, los datos de la base de datos son también libres. Para finalizar, la plataforma donde se ejecuta es libre: Ubuntu + Apache HTTP + Mysql + PHP.
